#  Happy New Year 2026 Fireworks

HTML5 Canvas と JavaScript で作った  
**「一文字ずつ花火で表示される Happy New Year 2026」アニメーション** です。

クリック不要・自動再生で、  
花火が **H → a → p → … → 2 → 0 → 2 → 6** の順に表示され、  
最後の「6」のあとに少し間を空けて、また最初から繰り返します。

PC・スマホ両対応です。

---

##  デモ内容

-  花火が連続で自動発射
-  **1発の花火につき1文字**
-  文字花火＋通常の放射状花火を同時表示
-  「2026」表示後は、次の「H」まで少し間を演出
-  スマホ / タブレット / PC 対応
-  Retina（高解像度）対応
